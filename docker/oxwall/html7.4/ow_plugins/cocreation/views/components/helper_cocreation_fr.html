Tutoriel de la section "Co-Création"
La Co-Création est un espace collaboratif pour permettre aux utilisateurs de collaborer et partager des connaissances ou des données. L'espace est destiné à être utilisé par des petits groupes, où les utilisateurs sont invités à participer par le créateur de la salle. Il existe deux sortes de éspaces: une salle "connaissance" de Cocreation, où le groupe est invité à réfléchir sur l'Open Data pour répondre à des problèmes et rendre les données plus simples et accessibles, et une salle "données" de Co-Création, où il est possible de créer en mode collaboratif des jeux de données.
1
Co-Création

Dans une salle de Co-Création vous pouvez trouver:

Une salle "connaissance

Une salle "données"

Une salle multimédia

Un vue sur tous les jeux de données

2
Tous les jeux de données

Cette fonctionnalité permet de voir tous les jeux de données co-créés dans des salles de co-création. Pour chaque jeu de données vous pouvez:

  • Visualiser un jeu de données publié dans une calle de Co-Création,
  • Télécharger un jeu de données en format CSV,
  • Copier le URL du jeu de données,
  • Télécharger les notes du jeu de données en format html.
3
Créer une salle

To create a new room, just press the big "+" (Add) button on the bottom right, and insert the required information: Pour créer une nouvelle salle, cliquer sur le bouton "+" en bas à gauche de la page d'accueil de l'éspace de la co-création, puis indiquez les informations requises:

  • Nom de la salle
  • Sujet
  • Description
  • Période (attention c'est une simple estimation)
  • Objectif
  • Texte d'invitation
  • Membres initials
4
Explorer
Cette fonctionnalité permet d'ouvrir une salle.
5
Toutes les fonctionnalités
  • Elle permet de selectionner toutes les fonctionnalités comme créer un jeu de données, un datalet, des metadonnées, des notes, commencer une discussion, etc.
  • Elle montre les informations d'une salle

  • Elle permet de rajouter des membres à une salle.
