Tuto d'une salle de Co-Création
Cette salle permet aux membres de collaborer sur des feuilles de calcul partagées pour créer un jeu de données. Le processus est pris en charge en écrivant des notes partagées (avec un modèle qui aide la communauté à écrire toutes les informations pertinentes du processus) et aux métadonnées partagées, qui sont publiés avec le jeu de données. En outre, lorsqu'ils sont publiés pour d'autres utilisateurs du SPOD, plusieurs côntroles de validation sont utilisés pour montrer des avertissements pour les champs apparament remplis d'une façon non comforme. Lors de leur publication, le jeu de données peut être utilisé (sous l'onglet "jeux de données créés conjointement" lors de la première étape) par tous les utilisateurs du SPOD dans leurs discussions. Il y a deux types de pièces: la salle de données et la salle des médias; la couleur reconnaît le type de pièce, par exemple le vert identifie la salle de données et l'orange celui de la salle des médias. La salle multimédia est synchronisée avec la même pièce de l'application mobile SPOD. En utilisant SPOD Mobile, les utilisateurs ne partageront pas seulement le contenu (c'est-à-dire, les photos) mais les photos incluront également automatiquement la géolocalisation pour chaque photo.
1
Un scénario de synchronisation
Les données synchronisées et la salle des médias présentent les mêmes caractéristiques, contrairement à la salle des données, la salle des médias a une feuille de calcul partagée avec des colonnes fixes (Titre, Description, Image, Emplacement, Date, Utilisateur) appareil mobile.
SPOD Mobile est actuellement disponible sur le Google Play Market à l'adresse SPOD mobile app.

2
Fonctionnalités
  • jeu de données permet de construire une feuille de calcul partagée pour saisir des données, d'importer et d'exporter un document au format CSV.
  • Metadonnées permet d'ajouter des métadonnées pour l'ensemble de données.
  • Notes permet d'ajouter des notes au jeu de données (e.g., sources, bibliographie, etc...)
  • Salle de discussion et notifications : dans la salle de discussion, un nouveau message est notifié avec une icône avec le compteur du nombre de messages reçus et non-lus.
  • Datalets permet de créer une carte interactive ou d'autres graphiques (par exemple, tableau, histogramme, etc.) d'un jeu de données.
  • Indique les détails lorsqu'une salle est créée (par exemple, le nom, la description etc ... de la salle).
  • Permet de choisir le type d'affichage, c'est-à-dire que vous pouvez afficher une fonctionnalité à la fois (par exemple, uniquement la feuille de calcul) ou deux fonctionnalités en même temps (par exemple, la feuille de calcul et la discussion).
  • Permet d'inviter de nouveaux utilisateurs à la salle
  • PUBLIER LE JEU DE DONNES SUR LE SPOD Permet la publication du jeu de données créé pour tous les utilisateurs de la plate-forme SPOD.
  • IMPORTER DE SPOD permet d'importer les deux ensembles de données créés par tous les utilisateurs de la plate-forme SPOD que les ensembles de données stockés dans le portail CKAN.

3
Un tour rapide

A quick tour to create and publish a dataset:

  1. 1. Créer un jeu de données dans la feuille de calcul:
    1. Créer (ou importer) un nouveau jeu de données en collaboration
    2. Ajouter des notes au jeu de données (par exemple, les sources, etc.)
    3. Ajouter des métadonnées au jeu de données (telles que le titre, la description, les balises, etc.) pour faciliter sa recherche
    4. Le jeue de données créé peut être affiché dans un tableau (datalet)

  2. 2. Publiez le jeu de données:
    Cliquez sur le bouton PUBLIER LE JEU DE DONNES SUR LE SPOD pour mettre à disposition le nouveau jeu de données créé pour tous les utilisateurs de la plate-forme SPOD et pour le rendre disponible pour créer un datalet utile aux discussions de manière traditionnelle. Vous pourrez partager sur la page, par exemple, dans "Quoi de nouveau" ou dans une salle de discussion de l'Agora. Notez que chaque fois que le même jeu de données est publié, une nouvelle version est créée.

  3. 3. Afficher les graphiques avec les jeux de données créés:
    Le jeu de données créé sera affiché dans l'onglet "jeux co-créés" du tableau de bord de la plateforme SPOD.
