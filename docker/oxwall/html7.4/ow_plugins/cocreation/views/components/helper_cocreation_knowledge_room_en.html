Cocreation Knowledge room helper
The Cocreation knowledge room is meant to stimulate reflection on Open Data through three steps process of co-writing document (with a template) for Explore, Ideas, and Outcome>. Datalets can be created and annotated via post-it like comments (notified in real time to participants).
1
Features
  • Each shared document, Explore , Ideas and Results , allows the co-writing process.
  • Library allows to insert the datalet for any views with charts (e.g. interactive maps, etc...)
  • Datalets allows to create visualization, i.e charts (named datalets).
  • Info allows to show details of the room, (i.e name, description etc.).
  • It is possible to choose the type of visualization, that is, you can view a feature at a time (eg only the spreadsheet), or two features at the same time (for example, the spreadsheet and discussion).
  • (on the top right side) New users can be invited into the room.

2
Library
Each user can add datasets of an Open Data portal (for example, @CKAN) that will be present as a list in the tab of the Controllet
  • Add datalet tab allows to add a datalet in the Library. A wizard (i.e., a guide) will help to add a dataset. The list of datalets will be available when a new chart added(i.e., )

  • Library tab shows all created dataset previously.
3
Co-creation

In this room, to the left, you can edit documents collaboratively, in real time and a list of features are available:

  • Collaboration: it is possible to see changes being made by other people in real time and each user is identified by color and name.
  • Chat: a synchronous communication is available with other people who share the same document.
  • Export: allows to export a document in HTML format, plain text, Microsoft Word, PDF and an Open Standard Format (ODF).
  • Recording Revision: the document is saved at and participants can permanently save specific versions checkpoints at any time. A time slider Allows anyone to explore the history of the document in a very user-friendly way.
