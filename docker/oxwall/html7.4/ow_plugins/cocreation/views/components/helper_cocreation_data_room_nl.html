Cocreation Data and Media Rooms helper
This type of room allows members to collaborate on a shared spreadsheets to create dataset. The process is supported by writing shared notes (with a template that helps the community to write all the relevant information about the process) and shared metadata, that are published with the dataset. Moreover, several validation heuristics are used to show warnings on apparently wrong fields. When published, the dataset can be used (under “Co-created datasets” tab on the first step of the Controllet) by all SPOD users in their discussions. There are two types of rooms: the data and the media room; the color recognizes the type of room, i.e the green identifies the data room and the orange that of the media room The media room is synchronized to the same room of the SPOD mobile application. By using SPOD Mobile, the users will not only share content (i.e., photos) but photos will also include automatically geolocation for each photo.

A synchronization scenario
The synchronized data and media room show the same features, in contrast with data room, the media room has a shared spreadsheet with fixed columns (named Title, Description, Image, Location, Date, User) that are automatically filled by taking a picture from mobile device.

Features
  • Dataset allows to write a shared spreadsheet to enter data, to import and export a document in CSV format.
  • Metadata allows to add metadata for the dataset.
  • Note allows notes to teh dataset (e.g., fonts, bibliography etc...)
  • Discussion room and notifications: in discussion room, a new message is notified with an icon with the counter of number of received and unread message.
  • Datalets allows to create interactive map or others charts (e.g., table, histogram etc. ) of a dataset (named datasets).
  • Shows details when a created room (e.g., name, description etc... of the room)
  • Allows you to choose the type of display, i.e you can view a feature at a time (eg only the spreadsheet or two features at the same time (e.g. the spreadsheet and discussion).
  • allows to invite new users to the room
  • PUBLISH DATASET ON SPOD allows the publication of the dataset created for all users SPOD platform.
  • IMPORT FROM SPOD allows importing both datasets created by all users SPOD platform that datasets stored in the CKAN portal.

3
A quick tour

A quick tour to create and publish a dataset:

  1. 1. Create a dataset in the spreadsheet:
    1. Create (or import) a new dataset in collaboration
    2. Add notes to the dataset (e.g. sources etc.)
    3. Add metadata to the dataset (such as title, description, tags, etc ..) to facilitate the research
    4. The created dataset can be displayed in a chart (datalet)

  2. 2. Publish the dataset:
    Click the button PUBLISH DATASET ON SPOD to make available the new created dataset for all users of SPOD platform and to make it available to create a datalet for discussions in the traditional ways, that you can share on the What's new page, in the discussion room of the Agora, etc. Note that each time the same dataset is published again, a new version is created.

  3. 3. View charts with created datasets:
    The created dataset will be showed in the "FROM COCREATED" tab of the controllet of SPOD platform.
