Cocreation helper
The CoCreation is a collaborative space to guide users to collaborate in the construction of knowledge, data or media room. The space is meant to be used by small groups, where user are invited to participate by the creator of the room. There are the three kinds of rooms: the Cocreation knowledge room, where the group is invited to reflect on Open Data to answer problems and create awareness of data, a Cocreation data room, where it is possible to cocreate datasets and Cocreated media room, where it is possible to create media datasets synchronized with the SPOD mobile application.

2
All datasets

This button shows all the datasets cocreated in the data rooms. For each dataset you can:

  • Show the dataset published in the Cocreation data room,
  • Download the dataset as CSV file,
  • Copy (to be later pasted) the address (URL) of the dataset,
  • download the notes of the dataset as html file.
3
Create a room

To create a new room, just press the big "+" (Add) button on the bottom right, and insert the required information:

  • Name of the room
  • Subject
  • Description
  • Period: it is just an indication of the expected activity of the room, not a prescription.
  • Goal
  • Invitation text to be sent to the members invited
  • Initial members (you will be able to add more in the room)
